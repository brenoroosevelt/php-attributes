# PHP Attributes Extractor

Easy way to extract and handle PHP Attributes.

## Requirements

* PHP >= 8.0

## Install 

```bash
composer require brenoroosevelt/php-attributes
```

## Usage
```php
<?php

```

## Run test suite

```bash
composer test
```

## Run analysis
* Test Suit
* Static Analyse
* Coding Standard

```bash
composer check
```
